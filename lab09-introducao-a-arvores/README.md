# LAB09-IntroducaoArvores

>**Árvores**

Composta por elementos menores, chamados de **Nodos**, possui como aplicação, dentre outras, o sistema de arquivamento que encontramos em múltiplos Sistemas Operacionais.


>**Nodo**
